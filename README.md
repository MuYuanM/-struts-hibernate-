javaweb小项目产品与分类增删改查管理系统
===
项目介绍
--
用框架struts与hibernate结合MVC的思想下做的一个产品与分类的对应关系的小项目

---

项目功能
---

- 产品的增加，删除，更改，查询

- 产品类别的增加，删除，更改，查询

- 产品与产品类别的对应关系，对产品进行分类
